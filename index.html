About

I was inspired to create this project as a way for non-sellers to easily sell their items through local and fast communication. I noticed that many people had items that they no longer needed or wanted, but they didn't have the time or resources to sell them through traditional means like garage sales or online marketplaces. I wanted to create a platform that would make it easy for anyone to sell their items locally and quickly.

  • Backend Development

    The backend development for this project involved designing and implementing an API, designing and implementing a database, and implementing security and authentication measures

    • API design and implementation
    • Database design and implementation
    • Security and authentication

  • Frontend Development

    The frontend development for this project involved designing and implementing a user interface, optimizing the design for responsive and mobile devices, and integrating with the backend API.

    • UI design and implementation
    • Responsive design and mobile optimization
    • Integration with backend API
